# Novelty Decision

Chosen thesis: embodied scaling laws are not universal; in contact-rich and safety-constrained regimes, additional model scale stops helping once the physical interaction bottleneck, not the representational bottleneck, dominates.

Why this thesis wins:
- It is supported by the strongest hostile priors rather than merely contradicting them.
- It avoids forbidden weak moves: no bigger model, no better data story, no benchmark-only claim, no generic uncertainty add-on, no active learning, no verifier, no planner-only synthesis, no RL escalation.
- It changes the central mechanism from "more scale" to "which physical regime controls the error curve."

What would make it fail:
- If the evidence only shows a lack of data, not a regime boundary.
- If manipulation and locomotion scaling are actually the same curve after normalization.
- If the proposed boundary cannot be tied to a concrete failure variable.

Decision: proceed with a regime-fracture paper built on literature evidence and a small runnable analysis of the collected corpus.
