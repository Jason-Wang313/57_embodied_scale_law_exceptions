# Novelty Boundary Map

## Not Novel

- Generic scaling laws.
- Foundation-model enthusiasm.
- Saying scale has limits.
- Contact-rich manipulation surveys.

## Plausible Contribution

A regime map that asks robotics scaling papers to report curves separately for contact-rich, safety-constrained, latency-sensitive, and morphology-fragmented regimes.

## V2 Boundary

The corpus evidence is not stable enough for a prevalence claim. It supports a workshop framing and falsifiable hypothesis.
